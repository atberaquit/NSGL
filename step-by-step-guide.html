Updating NetSuite DS Guides

Offshore SOP • One column • Mobile-first • Click images to zoom

Exemplar for this task: Create Cash Sale (guide and script)
Main Guide (e.g., Create Cash Sale [DS]) Guide Tips Guide (e.g., Create Cash Sale - Guide Tips)

Part 1: Check the Guide Script and the Guide

Guide Scripts are the single source of truth. Always compare the script vs actual guide behavior.

Step 1

Open the script of the guide assigned to you

Open the NSGL Global script file from SharePoint.

Open Script Link
Step 2

Look for the tab of the script assigned to you

Find the worksheet/tab for your specific guide.

Step 3

Open NetSuite and run the guide

Check where Guide Tips appear and compare it with the script.

Goal: Identify mismatches BEFORE you update the script and guides.

Part 2: Updating the Script

Update the script first so the guide work is aligned and verifiable.

Step 4

Update script using Create Cash Sale as the exemplar

Add GT apiName (if missing), fix tip positions, and renumber tips.

Open Exemplar Script
Add the following:
  • GT apiName (if not yet available)
  • Correct position of the Guide Tips
  • Number them accordingly
Step 5

Login to OGL and look for the guide you will update

Confirm the guide name and DS version before cloning.

Part 3: Create the Guide Tips Guide + Update the Main Guide

Follow the steps in order. Replace dux4kfwo with the Main Guide apiName.

Step 6

Duplicate the guide

Clone the guide so you can create a separate Guide Tips guide.

Step 7

Rename the cloned guide

Rename to: Name of the Guide - Guide Tips[DS]

Step 8

Copy the apiName and add it to the script

Paste it in the script under the GT apiName section.

Step 9

Prepare to update both guides

You will update the Guide Tips guide and the Main Guide.

  • Name of the Guide - Guide Tips[DS]
  • Name of the Guide [DS]
Step 10

Open the Guide Tips guide

Go to: Name of the Guide - Guide Tips[DS]

Step 11

Add a new Step 1 and add the following in Source

This is the controller step.

<p>[Controller]</p> <style type="text/css">div[class*="ns-cond-dux4kfwo"]{ visibility: hidden; } </style>
Take note: dux4kfwo is the apiName of the Main Guide.
Step 12

Add this custom CSS class

ns-popover-inner-hide

Step 13

Set the target to body

Target selector: body

Step 14

In all guide tips: remove step condition and add CSS class

Apply to every Guide Tip in the Guide Tips guide.

Remove step condition and add: ns-cond-dux4kfwo (Main Guide apiName)
Step 15

Save and Exit → go to Activation

Set autoload to the page where Guide Tips should appear.

Display this Guide in Autoload When the page has URL Matching /app/accounting/transactions/cashsale.nl
The URL is where the Guide Tips should appear.
Step 16

Guide Settings (Guide Tips guide)

Uncheck Sticky Guide

Step 17

Main Guide: Step 4 header — add the following in Source

Go to Name of the Guide [DS] and edit Step 4 header.

<style type="text/css">div.ns-cond-dux4kfwo { visibility: visible !important; } </style>
Step 18

In Step 4, remove the ns-cond custom class

Remove the ns-cond class from Step 4.

Step 19

Add ns-tip-zindex-999 custom class

Add ns-tip-zindex-999

Step 20

Save and Exit

Save the updated Main Guide.

Step 21

Test your guide and confirm the script is updated

Run the guide in NetSuite and verify Guide Tips + script alignment.

Done criteria:
  • Guide Tips appear on the correct URL
  • Script positions + numbering match actual behavior
  • GT apiName is recorded
