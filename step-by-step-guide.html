Updating NetSuite DS Guides

Apply the new guide structure to existing Deep Segmentation (DS) guides. Use Create Cash Sale as the exemplar.

Guide scripts are the single source of truth. Always validate and update the script before updating the guide(s).

You will update two artifacts: Main Guide + Guide Tips Guide.

REFERENCE Create Cash Sale (DS) — guide + script exemplar
Guide components

A DS guide consists of:
• The Main Guide (e.g., Create Cash Sale [DS])
• The Guide Tips Guide (e.g., Create Cash Sale - Guide Tips [DS])

Part 3: Create the Guide Tips and Update the Main Guide

Steps 6–20: Clone, configure Guide Tips, then update Main Guide
Expand/Collapse
6

Duplicate the guide

7

Rename the cloned guide

Rename to: Name of the Guide - Guide Tips[DS]

8

Copy the apiName and add it in the script under GT apiName

Paste the Guide Tips apiName into the script’s GT apiName section.

9

Prepare to update both guides

You will update: Name of the Guide - Guide Tips[DS] and Name of the Guide [DS].

10

Go to the Guide Tips guide

Open: Name of the Guide - Guide Tips[DS]

11

Add a new Step 1 and add the following in source

Replace dux4kfwo with the apiName of the Main Guide.

<p>[Controller]</p> <style type="text/css">div[class*="ns-cond-dux4kfwo"]{ visibility: hidden; } </style>

Note: dux4kfwo must be the Main Guide apiName.

12

Add custom CSS class

Add: ns-popover-inner-hide

13

Set the target to body

Target selector: body

14

For all Guide Tips, remove step condition and add CSS

Add custom class: ns-cond-dux4kfwo (Main Guide apiName).

15

Activation: Autoload on the correct URL

Use the URL where the Guide Tips should appear.

Display this Guide in Autoload When the page has URL Matching /app/accounting/transactions/cashsale.nl
16

Guide settings: disable Sticky Guide

Uncheck: Sticky Guide

17

Main Guide: Step 4 header — add the following in source

<style type="text/css">div.ns-cond-dux4kfwo { visibility: visible !important; } </style>
18

Main Guide Step 4: remove ns-cond custom class

Remove the ns-cond custom class from Step 4.

19

Main Guide Step 4: add ns-tip-zindex-999

Add custom class: ns-tip-zindex-999

20

Save and Exit

Save your changes.

Testing and Done Criteria

After updates, test in NetSuite and confirm script alignment.

  • Guide Tips appear on the correct URL
  • Tip order and placement match the updated script
  • GT apiName is recorded in the script
  • Main Guide Step 4 behavior is correct
